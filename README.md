# MTTO MVP (Frontend)

App MVP (solo frontend) para:
- Plan Semanal (matriz estilo Excel)
- Órdenes de Trabajo (OT)
- Horas Extra
- KPIs
- Admin

## Requisitos
- Node.js 18+ recomendado

## Ejecutar
```bash
npm install
npm run dev
```

Abrir: http://localhost:5173

## Nota
Este MVP usa datos mock en memoria. La integración a Firebase vendrá después de validar pantallas y flujo.
